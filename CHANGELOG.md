# Changelog

## 0.1.0 - 2026-05-05

- Initial release of `jortt`.
- Added typed OpenAPI operation enums for 126 operations across 15 tags.
- Added `JorttClient` facade with domain handles and high-frequency convenience methods.
- Added `RawClient` escape hatch.
- Added hybrid auth support via `AccessTokenSource` and `OAuthClient`.
- Added structured error mapping with preserved Jortt error payloads.
- Added UML package, architecture docs, auth guide, error guide, and endpoint matrix.
- Added CI workflow and crates.io publish readiness files.
